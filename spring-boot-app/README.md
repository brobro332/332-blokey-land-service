### 🛠 **프로젝트 환경**
- 언어: `Java 21`
- 프레임워크 : `Spring boot 3.5.0`
- 데이터베이스: `PostgreSQL`, `Elasticsearch`
- IDE: `IntelliJ`
- CSR : `React`
- 빌드 도구: `Gradle`
- ORM: `JPA`
- 동적쿼리 라이브러리: `QueryDSL`
- 배포: `OCI`
