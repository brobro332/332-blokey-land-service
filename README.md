# 🌿 java-co-working

### **🐶 개요**
개발자 프로젝트 관리를 위한 웹 사이트

### ⏱ **개발 기간**
- 2024-07-27 ~

### 🛠 **프로젝트 환경**
- 언어: `Java 17`
- 프레임워크 : `Spring boot 3.3.2`
- 데이터베이스: `Oracle 11g`
- IDE: `IntelliJ`
- CSR : `React`
- 빌드 도구: `Gradle`
- ORM: `JPA`
- 동적쿼리 라이브러리: `QueryDSL`
- 배포: `OCI`

```bash
java-co-working
├── Common
│   ├── WebConfig
│   ├── QueryDSL
│   └── CommonException
├── Project
│   └── Project CRUD API
├── Task
│   └── Task CRUD API
├── Team
│   └── Team CRUD API
├── Member
│   └── Member CRUD API
└── Milestone
    └── Milestone CRUD API
```
